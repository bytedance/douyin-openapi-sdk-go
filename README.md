# Douyin OpenAPI SDK for Go
本项目是抖音开放平台OpenAPI SDK 的Go语言实现。


## 要求
您需要确保本地安装的 go 环境版本大于等于 1.12.0.

## 安装
你可以使用 go mod 来管理你的依赖
```
go get github.com/bytedance/douyin-openapi-sdk-go
```
